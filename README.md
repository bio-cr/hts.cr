Crystal bindings to [HTSlib](https://github.com/samtools/htslib).

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     htslib:
       github: kojix2/htslib
   ```

2. Run `shards install`

## Usage

```crystal
require "htslib"
```
